# 项目介绍
本项目为一个AI智能文档产品
具体的功能包括：
1）知识库管理，知识库文档的增删查；  \
2）页面管理，页面的增删改查；  \
3）页面编辑；  \
其中，页面由封面、标题、块组成，块包括标题块、文本块、图像块；  \
对块，可以进行增删改查  \
4）AI功能，包括：文字生成、本地知识库问答
